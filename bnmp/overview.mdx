# 🚔 Mandados de Prisão — Visão Geral

A funcionalidade de **busca de mandados de prisão** da Predictus permite consultar, de forma padronizada, os registros associados a um **CPF** em bases oficiais.  
O retorno inclui **identificação da pessoa**, **dados do mandado** e **informações do órgão judiciário** responsável.

## 🔗 Como funciona o acesso

- A Predictus consulta **fontes oficiais** (BNMP/CNJ) e retorna os dados de forma estruturada.
- A chamada é **autenticada por token (Bearer)** e aceita um **CPF** por requisição.
- Respostas são **normalizadas** (padrões de datas, campos e enumerações), facilitando a integração.

---

## 🔐 Requisitos

- **Token de acesso** (Bearer)
- **CPF** válido (11 dígitos; pontuação opcional)
- Conectividade com a API Predictus

---

## ⚙️ Fluxo de consulta

1. **Envie a requisição** com o CPF para o endpoint de busca.  
2. A API retorna um **array** de mandados (ou `204` quando não houver registros).  
3. Utilize o `numeroProcesso` (quando presente) para correlacionar com outros recursos de processos da plataforma, se necessário.
