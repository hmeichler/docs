# 📦 Buscar Resultado dos Autos

Este endpoint retorna o status atual da solicitação iniciada com `buscarPorNumero`.  
Quando finalizado com sucesso, disponibiliza os links para download dos documentos.

---

## 🔗 Endpoint

```
POST https://api.predictus.com.br/predictus-api/processos/judiciais/autos/buscarResultado
```

---

## 🧾 Corpo da requisição (exemplo)

```json
{
  "id": "892d7822-ea43-4209-90f9-a66f8b983f7b",
  "sistema": "ESAJ-TJSP"
}
```

---

## 🔄 Status possíveis

| Status | Descrição                                |
|--------|--------------------------------------------|
| `PROCESSING` | Consulta em andamento               |
| `OK`         | Autos recuperados com sucesso        |
| `BAD CREDENTIALS` | Login/senha/OTP inválidos     |
| `SPECIAL ERROR` | Processo em segredo de justiça  |
| `UNKNOWN ERROR` | Falha genérica no tribunal       |

---

## 📄 Notas

- Os links retornados ficam disponíveis por até **24 horas**.
- Os arquivos são PDFs extraídos diretamente do portal judicial oficial.
