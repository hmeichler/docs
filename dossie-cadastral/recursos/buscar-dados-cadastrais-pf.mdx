# 🔎 Dossiê Cadastral por CPF

Este endpoint retorna o **perfil cadastral consolidado** de uma pessoa física a partir de um CPF, agregando dados da Receita Federal e fontes cadastrais com identificação completa, situação do documento, contatos, endereços, renda presumida, vínculos societários e histórico profissional.

A resposta é um **objeto único** — o CPF identifica unicamente a pessoa.

---

## 🔗 Endpoint

```
POST https://api.predictus.com.br/predictus-api/pessoas/fisicas/dadosCadastrais
```

---

## 🧾 Corpo da requisição

```json
{
  "cpf": "12345678901"
}
```

---

## 📚 Campos

| Campo | Obrigatório | Tipo | Descrição |
|-------|-------------|------|-----------|
| `cpf` | Sim | string | CPF a consultar — exatamente 11 dígitos numéricos, validado pelo algoritmo de CPF |

---

## 📥 Exemplo de cURL

```bash
curl --location 'https://api.predictus.com.br/predictus-api/pessoas/fisicas/dadosCadastrais' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}' \
--data '{
  "cpf": "12345678901"
}'
```

---

## 🧾 Códigos de resposta

| Código | Significado |
|--------|-------------|
| `200` | Sucesso — perfil cadastral encontrado |
| `204` | Nenhum dado cadastral encontrado para o CPF |
| `400` | Erro de validação no payload (CPF inválido ou ausente) |
| `401` | Token inválido ou expirado |
| `500` | Erro interno do servidor |

---

## 🛡️ Validações

- `cpf`:
  - Não nulo
  - Exatamente 11 dígitos numéricos
  - Validação de CPF formal (algoritmo de dígitos verificadores)

---

## 💡 Dica

> Antes de usar qualquer dado da resposta, verifique `situacaoCadastralRF`. CPFs com situação diferente de `REGULAR` (ex: `SUSPENSA`, `CANCELADA DE OFÍCIO`) podem indicar inconsistências cadastrais e merecem tratamento diferenciado no seu fluxo.
