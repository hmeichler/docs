# 🔎 Dossiê Cadastral por CNPJ

Este endpoint retorna o **perfil cadastral consolidado** de uma pessoa jurídica a partir de um CNPJ, agregando dados da Receita Federal com razão social, situação cadastral, atividade econômica, contatos, endereços, quadro societário, histórico de funcionários e dívidas federais.

A resposta é um **objeto único** — o CNPJ identifica unicamente a empresa.

---

## 🔗 Endpoint

```
POST https://api.predictus.com.br/predictus-api/pessoas/juridicas/dadosCadastrais
```

---

## 🧾 Corpo da requisição

```json
{
  "cnpj": "71676316000146"
}
```

---

## 📚 Campos

| Campo | Obrigatório | Tipo | Descrição |
|-------|-------------|------|-----------|
| `cnpj` | Sim | string | CNPJ a consultar — exatamente 14 dígitos numéricos, validado pelo algoritmo de CNPJ |

---

## 📥 Exemplo de cURL

```bash
curl --location 'https://api.predictus.com.br/predictus-api/pessoas/juridicas/dadosCadastrais' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}' \
--data '{
  "cnpj": "71676316000146"
}'
```

## 🧾 Códigos de resposta

| Código | Significado |
|--------|-------------|
| `200` | Sucesso — perfil cadastral encontrado |
| `204` | Nenhum dado cadastral encontrado para o CNPJ |
| `400` | Erro de validação no payload (CNPJ inválido ou ausente) |
| `401` | Token inválido ou expirado |
| `500` | Erro interno do servidor |

---

## 🛡️ Validações

- `cnpj`:
  - Não nulo
  - Exatamente 14 dígitos numéricos
  - Validação de CNPJ formal (algoritmo de dígitos verificadores)

---

## 💡 Dica

> Antes de usar qualquer dado da resposta, verifique `situacaoCadastral`. CNPJs com situação `BAIXADA`, `INAPTA` ou `NULA` indicam que a empresa não está em operação regular — combine com `dividas.total` para avaliar exposição a passivos federais.
