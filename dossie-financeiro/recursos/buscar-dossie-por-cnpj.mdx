# 🔎 Dossiê Financeiro por CNPJ

Este endpoint retorna o **perfil financeiro consolidado** de uma pessoa jurídica a partir de um CNPJ, agregando dados de bureaus de crédito (SPC/Serasa) com identificação cadastral, quadro societário, administradores, capital social, faturamento presumido, score, pendências, protestos, CCF, consultas recentes e dívida ativa.

A resposta é um **objeto único** (não uma lista) — o CNPJ identifica unicamente a empresa.

---

## 🔗 Endpoint

```
POST https://api.predictus.com.br/predictus-api/pessoas/juridicas/pendenciasFinanceiras
```

---

## 🧾 Corpo da requisição

```json
{
  "cnpj": "28942188000106"
}
```

---

## 📚 Campos

| Campo | Obrigatório | Tipo | Descrição |
|-------|-------------|------|-----------|
| `cnpj` | Sim | string | CNPJ a consultar — exatamente 14 caracteres alfanuméricos (12 alfanuméricos + 2 dígitos), validado pelo algoritmo de CNPJ |

---

## 📥 Exemplo de cURL

```bash
curl --location 'https://api.predictus.com.br/predictus-api/pessoas/juridicas/pendenciasFinanceiras' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {accessToken}' \
--data '{
  "cnpj": "28942188000106"
}'
```

## 🧾 Códigos de resposta

| Código | Significado |
| ------ | ----------- |
| `200`  | Sucesso — perfil financeiro encontrado |
| `204`  | Nenhum dado financeiro encontrado para o CNPJ |
| `400`  | Erro de validação no payload (CNPJ inválido ou ausente) |
| `401`  | Token inválido ou expirado |
| `500`  | Erro interno do servidor |

---

## 🛡️ Validações

- `cnpj`:
  - Não nulo
  - Exatamente 14 caracteres no formato `[0-9A-Za-z]{12}[0-9]{2}`
  - Validação de CNPJ formal (algoritmo de dígitos verificadores)

---

## 💡 Dica

> Combine `situacaoCadastral` (status na Receita) com `situacaoEspecial` (recuperação judicial, falência) para identificar empresas em estresse antes mesmo de avaliar `pendencias`. Empresas com `situacaoEspecial` preenchida geralmente exigem análise jurídica complementar — veja [Dossiê de Recuperação Judicial](../../dossie-rj/overview).
