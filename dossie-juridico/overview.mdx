# 📂 Dossiê Jurídico

O Dossiê Jurídico da Predictus é a porta de entrada para acessar o **histórico judicial completo** de uma pessoa, empresa ou conjunto de processos com características específicas.

Este módulo concentra os principais endpoints de consulta judicial, permitindo buscas por:

- CPF ou CNPJ de uma parte
- Nome completo ou parcial
- Número do processo (NPU/CNJ)
- Dados do advogado (nome, CPF ou OAB)
- **Critérios processuais personalizados** (valor da causa, assuntos, status, juiz, julgador etc.)

Com base nesses inputs, a Predictus retorna **dados estruturados** sobre os processos encontrados, como:

- Capa processual
- Polo das partes (ativo/passivo)
- Status padronizado (`StatusPredictus`)
- Ramo do Direito
- Movimentações e decisões (quando disponíveis)

---

## 🧠 Quando usar o Dossiê Jurídico?

Use este produto quando você quiser:

- Validar o histórico judicial de um candidato, fornecedor, cliente ou parceiro
- Automatizar due diligence jurídica
- Analisar riscos judiciais associados a um CPF/CNPJ
- Mapear carteiras processuais de uma pessoa ou empresa
- **Explorar e segmentar processos por critérios objetivos**, como valor, assunto, status ou tribunal
- Conectar sua operação à maior base judicial do Brasil via API
