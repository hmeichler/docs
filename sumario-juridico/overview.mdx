# 📊 Sumário Jurídico

O **Sumário Jurídico** oferece uma visão consolidada dos processos judiciais relacionados a uma pessoa física, empresa ou advogado, com base nos mesmos filtros do Dossiê Jurídico.

Ao invés de retornar todos os processos individualmente, este módulo fornece **estatísticas agregadas**, ideais para uso em dashboards, relatórios ou para análises rápidas de risco e exposição judicial.

---

## 🧠 Quando usar o Sumário Jurídico?

Use este produto quando você quiser:

- Obter um **panorama quantitativo** dos processos de uma parte
- Apoiar decisões estratégicas com **visões executivas**
- Construir painéis e indicadores jurídicos
- Evitar custos com o processamento completo de processos
